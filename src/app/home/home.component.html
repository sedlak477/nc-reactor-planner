Welcome to Nuclearcraft Reactor Planner

This website is a collection of tools that help you design your Nuclearcraft reactors!

Why?

I know that there already exists an excellent tool written by hellrage over on his GitHub. In fact that tool inspired me to attempt to create this website, so you should definitely check it out, it also supports actively cooled reactors!

This project is mainly for me to get more comfortable with Angular and web development in general.
